Project Overview

This project focused on developing a financial model and interactive dashboard to project revenue, costs, and profitability for Meenk, a mobile-first social platform. The model was designed to be assumption-driven, allowing users to input active users and immediately visualize outcomes across monetization through AdMob and infrastructure costs such as DevOps and APIs.

Our team built a scalable and defensible framework that supports conservative, expected, and aggressive growth scenarios. The final deliverable was intended to be both investor-ready and operationally useful, giving founders clear visibility into unit economics, break-even points, and overall financial sustainability.

Key Deliverables

  • Google Sheets financial model with dynamic inputs for active users, engagement, monetization, and cost drivers
  • Scenario-based projections across conservative, expected, and aggressive cases
  • Break-even analysis and profitability outputs
  • Interactive dashboard for revenue, cost, and margin visualization
  • Documentation of assumptions, formulas, and modeling logic
  • Final client presentation summarizing insights and recommendations
